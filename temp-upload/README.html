Weather Nemo

Weather Nemo is a simple weather app that tells the weather of a city through weatherapi.com's API.

Tech Stack

  • HTML
  • SCSS
  • JavaScript
  • React.js
  • Material UI
  • Netlify
  • React Query
  • Axios HTTP

How To Use?

Just select the desired city from the input form.

Contact

This is just a personal project of mine. It's not perfect and it's not meant to be relied on to get your daily weather. You can contact me on Twitter @HyphenFrox.
